Core Data Processing Application
A Java-based console application that processes structured CSV and JSON datasets, performs data cleaning and transformation, and generates analytical reports.

Features
1. CSV and JSON file ingestion
2. Data cleaning and validation
3. Data transformation pipeline
4. Console-based report generation
5. Modular and extensible architecture

Design Principles Used
1. Object-Oriented Programming (OOP)
2. SOLID Principles
   
Design Patterns:
1. Facade
2. Strategy
3. Template Method
4. Singleton

Testing:
1. JUnit tests implemented for core components
2. Validates data processing and transformation logic

Technologies:
1. Java
2. File Handling (CSV, JSON)
3. JUnit

Project Structure:
1. src/ → Main application code
2. test/ → Unit tests
3. nbproject/ → NetBeans configuration
